# 核心特性

SSH Terminal 提供了丰富的功能，旨在提升远程服务器管理的效率和体验。

---

## 多会话 SSH 终端管理

### 终端模拟器
- 基于 xterm.js 5.5 的现代化终端模拟器
- 支持 ANSI 颜色和光标控制
- 完整的 VT100/VT220 兼容性
- 支持 256 色和真彩色

### 多会话支持
- 同时管理多个 SSH 会话，终端标签页快速切换
- 会话持久化，支持会话分组和管理

### 终端主题
内置 8 种精美的终端主题：
- **One Dark** - 深色主题，适合开发者
- **Dracula** - 优雅的紫色主题
- **Nord** - 北欧风格，柔和舒适
- **Tokyo Night** - 深夜主题，护眼舒适
- **Monokai** - 经典的代码编辑器主题
- **GitHub Light** - 明亮清新的浅色主题
- **Solarized Light** - Solarized 浅色变体
- **Solarized Dark** - Solarized 深色变体

暂时不支持自定义主题。

## AI 智能助手

### 多 AI Provider 支持
支持多个主流 AI 服务提供商：
- **OpenAI 兼容格式** - GPT-4、GPT-3.5、DeepSeek、Qwen
- **Ollama** - 本地模型部署

### 智能功能
- **命令补全** - AI 智能提示和建议
- **命令解释** - 解释复杂命令的含义
- **自然语言转命令** - 用自然语言描述，AI 转换为命令
- **错误诊断** - 分析错误信息，提供解决方案
- **最佳实践建议** - 推荐更优的命令和操作方式

### AI 缓存
- 智能缓存 AI 响应，提升响应速度，减少重复请求，节省成本，本地存储，保护隐私

## SFTP 文件管理

### 双面板文件浏览器
- 类似 Total Commander 的双面板设计
- 本地和远程文件并排显示
- 拖拽上传和下载
- 文件和目录操作便捷

### 文件操作
- 上传、下载文件和目录
- 创建、删除、重命名文件和目录
- 修改文件权限和所有者
- 批量操作支持

### 高级功能
- 大文件传输优化
- 传输队列管理

---

## 云同步

### 数据同步
- SSH 会话配置云端备份
- 用户资料同步

### 账户系统
- 账号注册和登录，JWT Token 认证
- Token 自动刷新，多设备登录支持

### 同步策略
- 实时同步，手动同步
- 冲突解决机制，同步历史记录

---

## 用户认证系统

### 安全认证
- 邮箱注册和登录，密码哈希存储（Argon2）
- JWT Token 认证，Refresh Token 机制

### 数据加密
- AES-256-GCM 加密存储，密码和密钥加密，敏感信息保护

## 快捷键系统

### 预设快捷键
常用操作的快捷键绑定：
- `Ctrl+N` / `Cmd+N` - 新建连接
- `Ctrl+W` / `Cmd+W` - 关闭标签
- `Ctrl+Shift+K` / `Cmd+Shift+K` - 快速连接
- `Ctrl+Shift+A` / `Cmd+Shift+A` - NL 转命令
- `Ctrl+Tab` - 切换到下一个标签
- `Ctrl+Shift+Tab` - 切换到上一个标签

### 自定义快捷键
- 完整的自定义支持
- 快捷键冲突检测
- 导入和导出快捷键配置
- 按功能分类管理
