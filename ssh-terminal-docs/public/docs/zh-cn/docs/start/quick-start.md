# 快速开始

本指南将帮助你在 5 分钟内快速上手 SSH Terminal。

---


>完成本指南后，你将能够：
> - ✅ 创建第一个 SSH 连接
>- ✅ 使用终端执行命令  
> - ✅ 使用 AI 智能助手
> - ✅ 管理 SFTP 文件
>

---

## 📦 前置准备

在开始之前，请确保：
1. 已安装 SSH Terminal（[安装指南](/docs/start/installation)）
2. 有一台可访问的 SSH 服务器
3. 拥有服务器的登录凭证（用户名和密码或 SSH 密钥）

---

## 🚀 第一步：启动应用

### Windows
双击桌面图标或从开始菜单启动 SSH Terminal。

### macOS
从 Applications 文件夹或 Launchpad 启动 SSH Terminal。

### Linux
从应用菜单或命令行启动 `ssh-terminal`。

---

## 🔗 第二步：创建 SSH 连接

### 方法一：使用快速连接

1. 点击主界面的 **"新建连接"** 按钮（或按 `Ctrl+N`）
2. 填写连接信息：
   - **名称**: 给连接起个名字（如 "我的服务器"）
   - **主机**: 服务器 IP 地址或域名（如 `192.168.1.100` 或 `example.com`）
   - **端口**: SSH 端口（默认 `22`）
   - **用户名**: 登录用户名（如 `root` 或 `ubuntu`）
   - **认证方式**: 选择密码或密钥
     - **密码**: 输入登录密码
     - **密钥**: 选择私钥文件
3. 点击 **"连接"** 按钮

### 方法二：使用快捷键

按 `Ctrl+N` 打开快速连接对话框， 输入信息进行连接


---

连接成功后，你将看到一个功能完整的终端窗口。

验证是否可用：

```bash
# 查看当前目录
pwd
# 列出文件
ls -la /
```

### 终端操作

- **新建标签页**: 按 `Ctrl+T`
- **关闭标签页**: 按 `Ctrl+W`
- **切换标签页**: 按 `Ctrl+Tab` 或点击标签
- **复制**: 选中文本后，点击鼠标右键，选择**复制**
- **粘贴**: 按 `Ctrl+V`或者点击鼠标右键，选择**粘贴**

---

## 🤖 第四步：使用 AI 智能助手

SSH Terminal 内置 AI 智能助手，可以帮助你更高效地工作。

### 打开 AI 助手

点击侧边栏的 **"AI 助手"** 图标或按 `Ctrl+Shift+I`。

### 功能演示

#### 1. 命令解释（Ctrl+Shift+E）

如果快捷键没有生效，在设置中进行修改

选择终端的文本，AI 会解释它的含义：
```
用户:选择 "`ps aux | grep nginx`"

AI: 这个命令用于查找运行中的 nginx 进程：
- ps aux: 显示所有进程的详细信息
- |: 管道，将前一个命令的输出传递给下一个命令
- grep nginx: 过滤包含 "nginx" 的行
```

#### 2. 自然语言转命令(Shist+#)

用自然语言描述你想要做的事情：
```
用户: 查看系统内存使用情况

AI: 推荐命令：
free -h
```

#### 3. 错误诊断(Ctrl+Shift+A)

遇到错误时，让 AI 帮你分析：
```
用户: Permission denied when trying to delete a file

AI: 可能的原因和解决方案：
1. 文件权限不足，尝试使用 sudo
2. 文件被其他进程占用，检查并关闭
3. 文件系统只读，检查挂载选项
```

### 配置 AI Provider

首次使用时，需要配置 AI Provider：

1. 设置界面点击 AI Tab ，点击添加新服务按钮进行AI 配置
2. 选择 服务形式，输入相关信息，点击添加
3. 找到对应的AI 服务。配置API Key，BaseUrl 模型名称等等
4. 最后上的保存配置持久化

## 📁 第五步：管理文件

SSH Terminal 集成了 SFTP 文件管理功能。

### 打开 SFTP 管理器

点击侧边栏的 **"文件管理"** 图标。

### 基本操作

#### 上传文件

1. 在本地面板选择文件
2. 点击上方的上传按钮，或者快捷键 `Ctrl+U`

#### 下载文件

1. 在远程面板选择文件
2. 点击上方的下载按钮，或者快捷键 `Ctrl+D`

#### 文件操作

- **创建文件夹**:  点击文件面板上方的"新建文件夹"图标
- **删除**: 选中文件 → 点击文件面板上方的"删除"图标
- **重命名**: 右键 → 点击文件面板上方的"重命名"图标

---

## 🎨 第六步：个性化设置

### 切换主题

1. 点击右上角的 **"设置"** 按钮
2. 选择 **"外观"** → **"主题"**
3. 选择你喜欢的主题（如 Light、Dark、System 等）

### 配置快捷键

1. 点击 **"设置"** → **"快捷键"**
2. 修改你想要的快捷键
3. 点击 **"保存"**

### 其他设置

- **字体**: 设置终端字体和大小
- **光标**: 选择光标样式和颜色
- **滚动**: 设置滚动行数
- **声音**: 启用/禁用提示音

---

## 💾 第七步：会话管理

创建会话的会自动保存到本地的Sqlite文件持久化：

1. 点击侧边栏的 **"会话管理"** 图标
2. 你会看到所有已保存的会话
3. 点击会话卡片的连接按钮可以进行连接


---

## 📝 第八步：录制操作（可选）

SSH Terminal 支持录制终端操作。

### 开始录制

1. 点击上方工具栏的 **"开始录制"** 按钮
2. 开始执行你的操作
3. 点击 **"暂停"** 暂停录制
4. 点击 **"停止"** 结束录制

### 查看录制

1. 点击 **"录制管理"** 图标
2. 选择录制文件
3. 点击 **"导出视频"** ，点击开始导出，保存到本地
