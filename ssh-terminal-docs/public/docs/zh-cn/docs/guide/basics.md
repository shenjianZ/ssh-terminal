# 基础教程

本教程将帮助你快速了解 SSH Terminal 的基本功能和操作方法。

---

## 界面概览

SSH Terminal 的主界面分为以下几个区域：

### 侧边栏
侧边栏位于界面左侧，提供快速导航功能，包括：
- **终端** - 管理多个 SSH 终端连接
- **会话管理** - 查看和管理已保存的 SSH 会话配置
- **AI 助手** - 访问 AI 智能助手功能
- **文件管理** - 打开 SFTP 文件管理器
- **录制管理** - 查看和管理录制的终端操作

你可以使用 `Ctrl+B` 快捷键切换侧边栏的显示与隐藏。

### 终端区域
界面中央是主要的终端显示区域，支持多标签页管理。每个标签页代表一个独立的 SSH 连接会话。
终端区域上方有工具栏，提供常用功能的快捷访问：
- 新建连接
- 会话管理
- 开始/停止录制、录制管理

## 基本操作流程

### 创建 SSH 连接

有两种方式可以创建 SSH 连接：

#### 快速连接
按 `Ctrl+N` 快捷键打开快速连接对话框，填写以下信息：
- **主机地址** - 服务器 IP 地址或域名
- **端口** - SSH 端口（默认 22）
- **用户名** - 登录用户名
- **认证方式** - 选择密码或密钥认证
- **密码或私钥** - 输入密码或者私钥路径

快速连接不会保存配置，适合临时使用。

#### 创建持久化会话
在会话管理页面点击"新建会话"，填写连接信息后会话将被保存到本地数据库，方便重复使用。

### 连接管理

连接成功后，你可以：
- **新建标签页** - 按 `Ctrl+T` 创建新的终端标签页
- **关闭标签页** - 按 `Ctrl+W` 关闭当前标签页
- **切换标签页** - 按 `Ctrl+Tab` 切换到下一个标签页，按 `Ctrl+Alt+Tab` 切换到上一个标签页

## 终端使用基础

### 常用操作

- **复制文本** - 选中文本后使用鼠标右键菜单复制
- **粘贴文本** - 按 `Ctrl+V` 或使用鼠标右键菜单粘贴
- **查找文本** - 按 `Ctrl+F` 在终端中搜索文本
- **清屏** - 按 `Ctrl+L` 清除终端屏幕内容

### 字体调整

你可以实时调整终端字体大小：
- **放大字体** - 按 `Ctrl+=`
- **缩小字体** - 按 `Ctrl+-`
- **重置字体** - 按 `Ctrl+0` 恢复默认大小

字体缩放会累积，快速按多次会连续放大或缩小。

## 主题切换

SSH Terminal 提供多种终端主题供选择：

### 切换步骤

1. 点击右上角的"设置"按钮
2. 选择"终端" → "主题"
3. 从列表中选择你喜欢的主题

### 支持的主题

- One Dark - 深色主题，适合长时间使用
- Dracula - 经典的深色主题
- Nord - 清新的深色主题
- Tokyo Night - 现代化的深色主题
- Monokai - 经典的代码编辑器主题
- GitHub Light - 明亮的浅色主题
- Solarized Light - 经典的浅色主题
- Solarized Dark - 经典的深色主题

## 快捷键基础

SSH Terminal 提供丰富的快捷键支持，提升工作效率。

### 全局快捷键

- `Ctrl+N` - 新建连接
- `Ctrl+,` - 打开设置
- `Ctrl+B` - 切换侧边栏

### 终端快捷键

- `Ctrl+T` - 新建标签页
- `Ctrl+W` - 关闭标签页
- `Ctrl+Tab` - 下一个标签页
- `Ctrl+Alt+Tab` - 上一个标签页
- `Ctrl+F` - 查找
- `Ctrl+L` - 清屏
- `Ctrl+V` - 粘贴

你可以在设置页面自定义所有快捷键。
