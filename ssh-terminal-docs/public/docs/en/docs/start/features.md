# Features

SSH Terminal provides rich features designed to improve the efficiency and experience of remote server management.

---

## Multi-Session SSH Terminal Management

### Terminal Emulator
- Modern terminal emulator based on xterm.js 5.5
- Supports ANSI colors and cursor control
- Full VT100/VT220 compatibility
- Supports 256 colors and true color

### Multi-Session Support
- Manage multiple SSH sessions simultaneously, quickly switch with terminal tabs
- Session persistence, supports session grouping and management

### Terminal Themes
Built-in 8 beautiful terminal themes:
- **One Dark** - Dark theme, suitable for developers
- **Dracula** - Elegant purple theme
- **Nord** - Nordic style, soft and comfortable
- **Tokyo Night** - Night theme, eye-friendly and comfortable
- **Monokai** - Classic code editor theme
- **GitHub Light** - Bright and fresh light theme
- **Solarized Light** - Solarized light variant
- **Solarized Dark** - Solarized dark variant

Custom themes are not currently supported.

## AI Intelligent Assistant

### Multiple AI Provider Support
Supports multiple mainstream AI service providers:
- **OpenAI Compatible Format** - GPT-4, GPT-3.5, DeepSeek, Qwen
- **Ollama** - Local model deployment

### Intelligent Features
- **Command Completion** - AI intelligent prompts and suggestions
- **Command Explanation** - Explain the meaning of complex commands
- **Natural Language to Command** - Describe in natural language, AI converts to command
- **Error Diagnosis** - Analyze error messages and provide solutions
- **Best Practice Recommendations** - Recommend better commands and operations

### AI Caching
- Intelligent caching of AI responses to improve response speed
- Reduce duplicate requests and save costs
- Local storage to protect privacy

## SFTP File Management

### Dual-Panel File Browser
- Dual-panel design similar to Total Commander
- Local and remote files displayed side by side
- Drag-and-drop upload and download
- Convenient file and directory operations

### File Operations
- Upload, download files and directories
- Create, delete, rename files and directories
- Modify file permissions and owners
- Batch operation support

### Advanced Features
- Large file transfer optimization
- Transfer queue management

## Keyboard Shortcut System

### Preset Shortcuts
Keyboard shortcut bindings for common operations:
- `Ctrl+N` / `Cmd+N` - New connection
- `Ctrl+W` / `Cmd+W` - Close tab
- `Ctrl+Shift+K` / `Cmd+Shift+K` - Quick connect
- `Ctrl+Shift+A` / `Cmd+Shift+A` - NL to command
- `Ctrl+Tab` - Switch to next tab
- `Ctrl+Shift+Tab` - Switch to previous tab

### Custom Shortcuts
- Complete customization support
- Keyboard shortcut conflict detection
- Import and export shortcut configurations
- Manage by function category
