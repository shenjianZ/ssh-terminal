## 完整的快捷键列表及生效范围

  ✅ 全局快捷键（所有页面生效）

  | 快捷键 | 功能       | 说明                                                        |
  |--------|------------|-------------------------------------------------------------|
  | Ctrl+N | 新建连接   | 快速创建新的SSH连接，自动导航到终端页面并打开快速连接对话框 |
  | Ctrl+, | 打开设置   | 导航到设置页面                                              |
  | Ctrl+B | 切换侧边栏 | 显示或隐藏侧边栏                                            |

  生效页面：/、/terminal、/sessions、/settings、/sftp

  ---
  ✅ 终端快捷键（仅终端页面生效）

  | 快捷键       | 功能         | 说明                                                                   |
  |--------------|--------------|------------------------------------------------------------------------|
  | Ctrl+T       | 新建标签页   | 复制当前终端连接，创建新标签页（如果没有活跃标签则打开快速连接对话框） |
  | Ctrl+W       | 关闭标签页   | 关闭当前标签页                                                         |
  | Ctrl+Tab     | 下一个标签页 | 切换到下一个标签页                                                     |
  | Ctrl+Alt+Tab | 上一个标签页 | 切换到上一个标签页                                                     |
  | Ctrl+F       | 查找         | 在终端中搜索文本                                                       |
  | Ctrl+L       | 清屏         | 清除终端屏幕内容                                                       |
  | Ctrl+V       | 粘贴         | 粘贴剪贴板内容                                                         |
  | Ctrl+=       | 放大字体     | 增加终端字体大小                                                       |
  | Ctrl+-       | 缩小字体     | 减小终端字体大小                                                       |
  | Ctrl+0       | 重置字体     | 重置字体大小为配置文件的默认值                                         |

  生效页面：/、/terminal

  特殊说明：
  - Ctrl+C 未设置为快捷键，因为它在终端中有特殊的中断信号含义
  - 字体缩放功能会累积，快速按多次会连续放大/缩小

  ---
  ✅ SFTP 快捷键（仅 SFTP 页面生效）

  | 快捷键 | 功能         | 说明                           |
  |--------|--------------|--------------------------------|
  | Ctrl+U | 上传文件     | 上传选中的本地文件到远程服务器 |
  | Ctrl+D | 下载文件     | 下载选中的远程文件到本地       |
  | F5     | 刷新文件列表 | 刷新当前文件列表               |

  生效页面：/sftp

  特殊说明：
  - 使用 F5 而不是 Ctrl+R，避免与终端的反向搜索功能冲突

  ✅ 会话管理快捷键（仅会话管理页面生效）

  |--------|----------|--------------------------|
  | Ctrl+K | 新建会话 | 在会话管理页面创建新会话 |

  生效页面：/sessions

  🎯 设计原则

  1. 避免冲突：不与系统快捷键或终端特殊功能冲突
  2. 符合直觉：使用常见的快捷键组合（如 Ctrl+N 新建、Ctrl+W 关闭）
  3. 页面隔离：快捷键只在相关页面生效，避免误触发
  4. 可配置：用户可以在设置页面自定义所有快捷键
  5. 自动迁移：应用启动时自动清理废弃的快捷键配置

  ---
  💡 使用提示

  - 复制文本：在终端中选中文本后，使用鼠标右键菜单或系统快捷键（如 Ctrl+C）复制
  - 快速连接：在任何页面按 Ctrl+N 都会自动导航到终端页面并打开快速连接对话框
  - 切换标签页：使用 Ctrl+Tab 和 Ctrl+Alt+Tab 快速在多个终端标签之间切换
  - 调整字体：使用 Ctrl+= 和 Ctrl+- 调整到合适的大小，Ctrl+0 重置为默认值
